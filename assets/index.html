How to create a stick shortcut
  1. Long-touch on Home screen or select [Add] in menu of Home screen.
  2. Select "Shortcuts" in "Add to Home screen" dialog.
  3. Select "StickyShortcut" in "Select shortcut" dialog.
  4. Select target application in application list.
  5. Select favorite icon from shortcut icons.

How to use sticky shortcut
  • By touching sticky shortcut on Home screen, target application will be launched.
  • If target application is not installed, detail page of Android Market will be opened.
  • Sticky shortcut will work again if you re-install target application.

Notes
  • Sticky shortcut will remain if target application is uninstalled.
  • Depending on Home application, all Sticky shortcuts may be removed on uninstalling StickShortcut application itself.

Troubleshooting
Please remove all sticky shortcuts and re-create them.
